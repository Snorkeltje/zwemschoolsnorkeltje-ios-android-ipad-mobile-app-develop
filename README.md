# Zwemschool Snorkeltje - Mobile App

Swimming school management mobile application for **Zwemschool Snorkeltje** (Bunschoten Spakenburg, Netherlands).

## Features

- **Customer App** (Light Theme - Dutch)
  - Lesson booking with fixed slot system
  - Punch card management
  - Child progress tracking (Level > Skill > Steps)
  - Payment via Stripe & punch cards
  - Chat with instructors
  - Automated waitlist

- **Instructor App** (Dark Theme)
  - Daily schedule management
  - Student progress updates
  - Offline-first with sync

## Tech Stack

- **Framework:** Flutter (iOS + Android + iPad)
- **State Management:** Riverpod
- **Navigation:** go_router
- **Backend:** Node.js/Express + PostgreSQL/Supabase
- **Payments:** Stripe
- **Push Notifications:** Firebase FCM
- **Offline Storage:** Hive

## Getting Started

```bash
flutter pub get
flutter run
```
